# 01 — Hello World

A simple polyglot "Hello World" example demonstrating Klyron's ability to run multiple programming languages from a single runtime.

## Files

| File         | Language   |
|-------------|------------|
| `hello.js`  | JavaScript |
| `hello.ts`  | TypeScript |
| `hello.py`  | Python     |
| `hello.go`  | Go         |
| `hello.php` | PHP        |
| `hello.rb`  | Ruby       |
| `hello.rs`  | Rust       |

## Run

```bash
klyron run hello.js
klyron run hello.ts
klyron run hello.py
klyron run hello.go
klyron run hello.php
klyron run hello.rb
klyron run hello.rs
```
