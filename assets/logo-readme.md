# Klyron Branding Assets

## Logo Files

- **logo.svg** (512x512) — Full logo with rounded square background, suitable for social cards, website headers, and README banners
- **logo-icon.svg** (256x256) — Compact icon version for favicons, app icons, and navigation

## Design

The Klyron logo combines an abstract "K" with a lightning bolt motif, symbolizing speed and power. The gradient transitions from indigo (6366f1) to purple (a855f7).

## Usage Guidelines

- Do not distort, rotate, or modify the logo proportions
- Use on dark or light backgrounds with sufficient contrast
- Preferred background: dark (#0f172a) or transparent
- Minimum clear space: 20% of the logo size on all sides

## Colors

| Name    | Hex     |

## Formats

SVG files are the source of truth. Raster exports (PNG, WebP) should be generated from SVGs when needed.
