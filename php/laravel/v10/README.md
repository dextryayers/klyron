# Klyron Laravel 10.x Stubs

Compatibility stubs for running Laravel 10.x applications with Klyron.

## Requirements

- PHP 8.1+
- Laravel Framework 10.x
- Klyron CLI

## Usage

```bash
klyron artisan serve
klyron artisan migrate
klyron artisan make:model User
```

## Supported Features

- Artisan CLI commands
- Routing (web + API)
- Eloquent ORM
- Blade templating
- Queues (sync and database)
- Mail (log and SMTP)
- Events and listeners
- Caching (file and database)
- Session (file and database)
- Validation
- Testing (PHPUnit + Klyron test runner)
