# Klyron Laravel 13.x Stubs

Compatibility stubs for running Laravel 13.x applications with Klyron.

## Requirements

- PHP 8.3+
- Laravel Framework 13.x
- Klyron CLI

## Usage

```bash
klyron artisan serve
klyron artisan migrate
```

## What's New in v13

- Latest Laravel framework improvements
- New first-party packages
- Enhanced security defaults
- Improved request handling
- Updated dependency requirements

## Klyron Integration

- Preview support for early adopters
- Full Artisan compatibility
- Built-in performance optimizations
- Deploy to Klyron with `klyron deploy`
