# Klyron Laravel 12.x Stubs

Compatibility stubs for running Laravel 12.x applications with Klyron.

## Requirements

- PHP 8.2+
- Laravel Framework 12.x
- Klyron CLI

## Usage

```bash
klyron artisan serve
klyron artisan migrate
```

## What's New in v12

- Improved default pagination
- New Artisan command structure
- Enhanced validation rules
- Performance improvements
- Updated minimum PHP version requirements

## Klyron Integration

- Full Artisan command compatibility
- Automatic environment detection
- Optimized for Klyron's serverless deployment
