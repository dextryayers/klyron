# Klyron

**Universal JavaScript runtime and package manager** — run Express, Hono, Koa, Elysia, Laravel, and any web framework natively.

## Installation

### npm (macOS / Linux / Windows)

```bash
npm install -g klyron
```

### curl (macOS / Linux)

```bash
curl -fsSL https://raw.githubusercontent.com/dextryayers/klyron/main/install.sh | bash
```

### Windows PowerShell

```powershell
iwr -useb https://raw.githubusercontent.com/dextryayers/klyron/main/install.ps1 | iex
```

### Cargo

```bash
cargo install klyron-cli
```

## Quick Start

```bash
klyron --version
klyron run hello.js
```
